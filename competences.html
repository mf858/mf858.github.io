Compétences

Projets d'étude

Projet 1

Création d'un jeu de Sudoku en langage C

Compétence : Etre capable de programmer un programme simple en respectant les règles du sudoku et en utilisant les differentes structures du langage C.

Projet 2

Création d'algorithmes en C pour la résolution du sudoku

Compétence : Etre capable d'imaginer des algorithmes de résolution et de les implémenter en C.

Projet 3

Production d'un systeme pour automatiser la documentation de programmes en C

Compétence : Etre capable de générer une documentation automatique d'un programme en C grace à un script PHP.

Projet 4

Création d'un diagramme UML et d'une base de donnée à partir du programme national du BUT

Compétence : Etre capable de créer une base de données à partir d'un texte complexe.

Projet 5

Création d'un site web pour repondre à la demande d'un client

Compétence : Etre capable de créer un site web à partir d'un cahier des charges.

Projet 6

Analyse de la RSE de l'entreprise Schneider

Compétence : Etre capable d'analyser la RSE d'une entreprise française ou étrangère grâce à des recherches sur internet.

Projet 7

Réalisation d'une application en java depuis un cahier des charges

Compétence : Etre capable de créer une application avec javafx pour répondre aux demandes d'un client

Projet 8

Recherche d'algorithmes pour résoudre le problème des 8 dames

Compétence : Etre capable d'imaginer et de mettre en place des systemes de resolution en Python.

Projet 9

Simulation d'une installation serveur sur une machine virtuelle

Compétence : Etre capable d'installer un serveur Apache pour stocker des pages web et sécuriser des documents.

Projet 10

Exploitation d'une base de données afin d'analyser des statistiques

Compétence : Etre capable de peupler une base de données grâce à des fichiers csv puis de l'exploiter afin d'en tirer des statistiques.

Projet 11

Gestion d'un projet, répartition des tâches entre les membres de l'equipe et diagramme de Gantt

Compétence : Etre capable de gérer efficacement un projet avec les membre d'une équipe pour être plus efficace.

Projet 12

Création d'un évènement de team building et de toute la communication autour : flyer, vidéo...

Compétence : Etre capable de créer un évenement de team building et de communiquer autour.
